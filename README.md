Develop a game where you have $1000 from the start. You bet fake money, you throw the dice and either lose the bet or you win and triple your bet.
The goal ov the game is winning as much money as possible and then quit playing the game while on top.

INSTRUCTIONS

1 - Create a variable named 'balance' and assign the value ov 1000. 

2 - Ask the user to bet.

3 - Read the bet into the variable bet.

4 - Ask the user for a guess between one and six (and tell the user to guess zero in order to quit). Read the guess into variable 'guess'.

5 - If the guess is zero, then quit the game.

6 - Pause the game for two seconds.

7 - Generate a random number between one and six; put the value in the variable 'dice'.

8 - Check if the guess matches the dice.

9 - If they don't: subtract the bet from the balance.

10 - If they do match: multiply the bet by three and add that ot the balance.

11 - If the balance is zero, quit the game.

12 - If the balance is positive, loop the game to be able to play more rounds.

13 - Add some text in order to better communicate with the user (e.g. "throwing the dice...", "congratulations", "you lost", "your balance is..." etc).

14 - Add comments to your code in order to better explain the different parts of it.
